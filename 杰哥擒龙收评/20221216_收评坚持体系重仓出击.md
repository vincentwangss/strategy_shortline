#  收评：坚持体系，重仓出击

**收盘点评：**

今天市场还是有不少能聊的，总的来说市场现在对高位的辨识度的反馈都还可以，基本没啥A杀的，都在不断拓展高度，还是比较良性的，不过量能还是不够主升，都还在轮动，所以继续围绕
**各个方向的核心博弈** 就好。  

**医药** 这边很多人看空，我昨天就说了资金介入那么深，疫情还在不断发展，不会直接就结束的，多看一些  **新逻辑新方向** ，这不  **中国医药**
和辉瑞一签协议，新的刺激点马上就来了，退烧药炒久了还是过渡到特效药了，毕竟阳了是真难受还是希望有更有效一些的药的，所以这两天资金就围绕这块发散了。今天中国医药竞价直接一字涨停，
**众生药业** 昨天炸板分歧，今天也是直接爆量开在昨天涨停价以上，都是明显加强的信号，所以直接围绕医药的  **辨识度参与**
就好了，除了众生就是新华制药亨迪药业这些，20cm的多瑞医药广生堂上海凯宝等也都表现不错，也都是可以跟随套利的。最近周五都喜欢炒药  **博弈周末**
，看到信号跟随上车就行，日内利润大一些周一低开也无妨的，不过后排还是要当心，还是容易回落的。

![](https://mmbiz.qpic.cn/mmbiz_png/TSg5ckQoYIP9NCqGTq6Ku1icNIdkLZeAicxcHiaH9yysXeldH9jn0PTyd1YmqSnGSr2ZNB6NEOzC0NeibXaV1qgJ8Q/640?wx_fmt=png)

**消费** 这边今天也还可以，早上被医药吸金有一波分歧，后面基本还是围绕几大核心在博弈。早盘比较强的是  **如意集团**
，绕开移动后这两天都很强，昨天那个炸板回封确实是很好的机会，今天也是继续加速了。有了如意的模板，  **人人乐**
今天也是在尝试绕异动了，低开高走走的还是比较稳的，尾盘的控盘也真的是精准，不多不少就差一分钱，正好不异动，那后面就还是有预期的。还有就是这波明牌的龙头
**西安饮食**
了，今天就是只能分歧低吸了，9.45分那波杀跌倒是个上车机会，这种明牌博弈不用打板要的就是日内性价比。除了这三个龙头外剩下的应该就是跟风了，昨天没打到西安想做个修复，结果二选一选错了真是有写遗憾，还是应该选更强的
**海南椰岛** 的，日内至少给了不错的修复，  **跨境通**
日内分时还是弱了，虽然图形觉得还行但终究还是没走出来，所以开盘没能主动上攻破均线也是回拉后离场了。还有个  **黑芝麻**
，早上计划想做反核的，可惜被抢到了平盘，性价比稍微差了点，不过后面有一波下杀到-5倒也算个机会，下午能反包上板还是超预期的，只能说这票就是股性好，资金还是有记忆的。

别的方向主要是  **地产和芯片** ，也都是有消息刺激的方向，那就是围绕核心做好轮动了。  **地产**
今天有陆家嘴的利好一字顶着，中交地产也尝试过二板，不过感觉买点还是在昨天，今天属于右侧盈亏比就一般，但还是有反复机会的。  **芯片**
这边周三爆发，周四加强，还是超我预期了，应该也在博弈经ji会议的利好，那么今天周五被别的方向吸金后早上大分歧，其实是不错的机会。不过标地上还是要选抗分歧的好，
**大港股份** 虽然地位高但走的不行，2板的易天股份和中晶科技也是，分时都是往下的不太喜欢，看下来就觉得1进2的  **中京电子**
不错，昨天烂板今天弱转强，明显扛住了板块的分歧，所以也是均线下方直接低吸上车，下午加强后率先拉升上板，强度很不错，所以也是继续加仓到了重仓，我还是会坚持分歧买核心的体系的~看看周末有啥消息吧，这个方向也是有回流预期的。
